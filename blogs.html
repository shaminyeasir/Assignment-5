Question-1: What are the different ways to select an element in the DOM?

To select an element in the DOM id, class, tag name can be used by using document.getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll() etc.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML returns and sets html content of an element, while innerText and textContent returns and sets only text content of an element.
innerText returns only visible text contents while textContent returns both visible and hidden text contents.

Question-3: What is event delegation in the DOM?

Event delegation is an advanced technique in Javascript which handles event listeners in a very efficient way by using the event bubbling property. In case of event delegation, only one event listener is used with the parent node instead of using several event listeners with the child nodes.

Question-4: What is event bubbling in the DOM?

In DOM, event bubbling is a concept where if a child element is triggered by an event, all elements from parent element to that child element got triggered at first top to bottom and then bottom to top.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element I can use document.createElement().
To add an element I can use appendChild().
To remove an element I can use removeChild() or remove().
